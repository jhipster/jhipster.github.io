Démarrage rapide avec JHipster

  1. Installez Java, Git et Node.js
  2. Installez JHipster npm install -g generator-jhipster
  3. Créez un nouveau répertoire et entrez dedans mkdir myApp && cd myApp
  4. Lancez JHipster et suivez les instructions à l'écran jhipster
  5. Modélisez vos entités avec JDL Studio et téléchargez le fichier jhipster-jdl.jdl résultant
  6. Générez vos entités avec jhipster import-jdl jhipster-jdl.jdl

Qu'est-ce que JHipster ?

+
/
/
=

JHipster est une plateforme de développement pour générer, développer et déployer rapidement des applications web modernes et des architectures de microservices. Nous supportons de nombreuses technologies frontend, y compris Angular, React et Vue. Nous avons même un support pour les applications mobiles avec Ionic et React Native ! Pour le backend, nous supportons Spring Boot (avec Java ou Kotlin), Micronaut, Quarkus, Node.js et .NET. Pour le déploiement, nous adoptons les principes cloud native avec Docker et Kubernetes. Le support de déploiement existe pour AWS, Azure, Cloud Foundry, Google Cloud Platform, Heroku et OpenShift.

Objectif

Notre objectif est de générer une application web complète et moderne ou une architecture de microservices, en unifiant :

  • Une pile serveur performante et robuste avec une excellente couverture de tests
  • Une interface utilisateur élégante, moderne et mobile-first avec Angular, React ou Vue + Bootstrap pour le CSS
  • Un workflow puissant pour construire votre application avec Webpack et Maven ou Gradle
  • Une architecture de microservices résiliente avec des principes cloud native à l'esprit
  • Une infrastructure as code pour que vous puissiez déployer rapidement sur le ☁️
